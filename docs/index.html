Introduction

Joytan (ジョイ単) is a free, small cross-platform desktop application that facilitates the process of making audio/textbook and helps people create their own original educatinal materials. It comes with some powerful features which are designed to be especially useful for language-learners.

Features

  • Creating MP3 audiobooks with your best-loved songs and sound effects
  • Full control of sequence and every timbre of your audiobook
  • Audiobook with lifelike speech using Amazon Polly (in 24 languages with 52 voices!)
  • Support free, fast and local Text-to-Speech alternatives
  • Creating textbook offline for free, easy to export to PDF
  • Designing original textbooks with basic HTML
  • Automatic language translation powered by Google Translation
  • Automatic visual aid (images) downloading for textbook using Google Image
  • Automatic online dictionary-lookup for language learning
  • Saving scripts in CSV files, ready to export to Anki
  • Editing scripts online even with people around the world via Google Sheets
  • Simple and tiny GUI, just 3 key dialogs!
